O que é CSS

CSS significa Cascading Style Sheets(ou Folhas de Estilo em Cascata).

É uma linguagem de estilização usada para:

  • Mudar cores de texto e fundo
  • Escolher fontes e tamanhos
  • Criar layouts responsivos
  • Adicionar animações e transições
  • Posicionar elementos na tela
    • Menus
    • Botões
    • Imagens
